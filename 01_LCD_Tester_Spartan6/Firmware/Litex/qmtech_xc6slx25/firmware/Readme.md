# Custom Firmware Development

Files required to create custom firmware:

1. **Build Folder:** Must already exist (generated by LiteX).
2. **Compiler:** RISC-V GCC toolchain.
3. **Linker Script:** `regions.ld` (generated by LiteX).
4. **Headers:** `generated/csr.h` and other LiteX-generated headers.
5. **Program Files:** Your source code (`.c` and/or `.h`).

# Simulation

Simulation directory that contains python file for generating .vcd simulation file.

## How to Run Guide

- Make sure litex environment already activated
- Head to simulation directory <code>cd simulation</code>

```bash
cd simulation
```

- Run the python build

```bash 
python top.py 
```

- .vcd simulation file will be generated inside <code>simulation</code> directory
- Open the .vcd file using <b>vaporview (vscode)</b> or <b>vgtkwave</b>
